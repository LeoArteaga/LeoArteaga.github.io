El proyecto de tercer semestre fue desarrollar un sitio web para consultar una base de datos de genes y operones usando HTML, CSS, JavaScript, PHP y MySQL. El sitio tenía que tener un nombre, logotipo y diseño hecho por nosotros y se iba a componer de varias páginas:

  • Una página principal donde se mostrara el logotipo, el nombre del sitio y una barra de búsqueda con opciones para buscar genes, operones o ambos
  • Una página de descargas/registro de usuarios donde se podría descargar los datasets de genes y operones o la base de datos completa y donde se podrían registrar los usuarios en otra base de datos ingresando los siguientes datos: first name, last name, institution, country, email, password. Con validaciones para que no se registrara el mismo usuario dos veces
  • Una página donde se mostrara información del autor del sitio (en este caso, de mí)

Respecto a la búsqueda, se valida que el usuario eligiera al menos una de la dos opciones de búsqueda: genes u operones, al situarse sobre la barra de búsqueda y comenzar a escribir se despliegan sugerencias de autocompletado y al buscar una palabra se manda a una página de resultados donde se muestran los resultados de la búsqueda en una tabla para genes y una tabla para operones. Al hacer click izquierdo sobre uno de los genes/operones se abre otra ventana que manda a la página con información sobre ese gen/operón, dentro de esta página también se puede descargar la información del gen/operón.

Este semestre hemos visto cosas más específicas de bioinformática como conexión a, navegación y búsqueda en bases de datos, lectura y quality control de secuencias genómicas y transcriptómicas, visualización, variant calling, entre otras.
